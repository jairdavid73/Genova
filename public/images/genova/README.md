# Imagenes de Genova

Pon aqui tus imagenes para el hover.

Ahora mismo `app/page.tsx` usa `/images/genova/placeholder.svg` para que no se rompa la vista.

Cuando agregues tus imagenes, puedes reemplazarla por rutas como:

- `/images/genova/imagen-1.jpg`
- `/images/genova/imagen-2.jpg`
- `/images/genova/imagen-3.jpg`

Puedes cambiar los nombres en `app/page.tsx` si tus archivos usan otros nombres.
